# Godot Git Workflow Tutorial Project
This repository accompanies my YouTube tutorial on using **Git and GitHub with a Godot Project**.

The goal of this project is to demonstrate a clean and practical Git workflow when starting and working on a real game project in Godot.

The repository contains a small Godot project used throughout the tutorial to demonstrate:
* Creating a repository
* Cloning a project
* Branch workflows
* Commits and history
* Pull requests
* Git LFS usage for game assets
* Adding Git to an existing project

## Requirements
This project was created using **Godot 4.6**.

Opening the project with **Godot 4.6 or newer** is recommended.<br>
Older versions of Godot 4.x may not open the project correctly.

To open this project you will need:
* Godot 4.6
* Git
* Git LFS (required if cloning the repository normally)

## Using This Project
You can clone the repository using either SSH or HTTPS.

Example (SSH)
```bash
git clone git@github.com:fat-earth-studios/tutorial-demo-repo.git
```
Example (HTTPS):
```bash
git clone https://github.com/fat-earth-studios/tutorial-demo-repo.git
```

After cloning, open the project in Godot:

`main-game-setup-tutorial/project.godot`
